Question-1: What are the different ways to select an element in the DOM?

There are so many ways to select an element in the dom. These are getElementById , getElementByClassName ,querySelector ,querySelectorAll , getElementByTagName etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Upon question- innerHTML gives all the html element inside selected id, innerText gives all the text inside selected id and textContent gives the text inside the tag but not returns tags.textContent returns hidden text also while innerText doesnot.

Question-3: What is event delegation in the DOM?

When event occurs in child element it bubbles up to parent element. so based on this problem instead of targetting child elements we can add single event handler on parent .so bubbling will not occur.

Question-4: What is event bubbling in the DOM?

In Browser when we set an event on any element the browser start capturing where the targetted element is. After capturing in the targetted phase capturing process jumps to targetted elements parent .If the parent also selected for targetting then through the child element and parent element bubbling occurs
